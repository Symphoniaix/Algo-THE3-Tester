# Algo-THE3-Tester
Algorithms THE3 Tester

How to run?
1. Place your the3.cpp file in the same directory with the tester.
2. ~$ make clean
3. ~$ make all
4. ~$ make run
(or simply, ~$ make all)

I have also added the matrix generator I have used to generate the inputs. Feel free to make it more randomized.
